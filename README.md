
# Eid Mubarak Interactive Greeting

This repository contains an interactive **Eid Mubarak** greeting webpage created by **Chowdhury IT Organisations**. The page features animated visuals, background music, and personalized messages to celebrate Eid.

## Features

- **Animated UI:** Glassmorphism-style card with floating orbs and confetti effects.
- **Audio Playback:** Background Eid song plays automatically when the greeting starts.
- **Step-based Greeting:** Sequentially shows multiple message steps with typing effect.
- **Personalized Touch:** Displays a profile image and final celebratory message.
- **Responsive Design:** Works on both desktop and mobile devices.

## Files

- `index.html` — Main HTML file with embedded JavaScript and CSS for the greeting.
- `assets/audio/Eid_Mubarak_Naat_2022.mp3` — Background audio for the greeting.
- `assets/images/` — Placeholder images for logos and profile.

## How to Run

1. Clone or download this repository.
2. Make sure `index.html` and `assets/` folder are together.
3. Open `index.html` in a modern web browser (Chrome, Firefox, Edge, etc.).
4. Click the **দেখুন** button to start the greeting and play the audio.
